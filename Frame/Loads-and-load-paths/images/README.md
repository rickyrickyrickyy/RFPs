# Frame subsystem — overview images

CAD views of the main frame and the three interface plates. Colour coding in the renderings:

| Colour | Hex | Component |
|--------|-----|-----------|
| Bright green | #26C16A | **Main frame** — welded structure (80×80×5 GB/T tubes, #8 C-brackets) |
| Greenish-blue / teal | #34E682 | **Loaded mount plate** — collection assembly; collection module and static side of extraction (juicer mashes oranges into) |
| Purple / magenta | #B341C3 | **Transmission mount plate** (formerly bearing mount plate) — part of **transmission subsystem**: yoke, two 35 mm shafts, two LMK35UU linear bearings, and this plate |
| Bluish-purple | #9075CD | **Drivetrain mount plate** — two gears, housed tapered roller bearings (gears on top), reducer spacer, Sumitomo reducer |
| Beige / light orange | #FFD691 | **Leveling mounts** — 6111K669 M24 100 mm Swivel Leveling Mount (McMaster), screwed into M24 tapped endplates on bottom of frame legs |

All three plates are mounted from the top onto the main frame and installed onto the two side rails.
