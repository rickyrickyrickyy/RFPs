# Transmission subsystem

The **transmission system** consists of the **yoke** (#2F9EBA), **two 35 mm drive shafts with shaft mounts** at both ends (#798C2E), and **two LMK35UU linear bearings** (#3582E4), all supported by the **transmission mount plate** (#B341C3) on the main frame.

## Function

Transmits **force from the gears** (drivetrain) **to the extraction/synchronisation system**, which **presses fruit against the collection system** mounted on the **collection system mount plate** (formerly static mount plate / loaded mount plate).
