# Transmission subsystem — images

CAD colour coding for the transmission system:

| Colour | Hex | Component |
|--------|-----|-----------|
| Turquoise / cyan | #2F9EBA | **Yoke** — moving part; connects to shafts; transmits force to extraction/synchronisation |
| Dark green / olive | #798C2E | **Two drive shafts** with **shaft mounts** at both ends (35 mm shafts) |
| Blue | #3582E4 | **Two LMK35UU linear bearings** — guide linear motion of shafts through the transmission mount plate |
| Magenta / pink | #B341C3 | **Transmission mount plate** — mounts to main frame; supports yoke, shafts, and bearings |

## Function

The transmission system transmits force from the **gears** (drivetrain) to the **extraction/synchronisation system**, which presses fruit against the **collection system** mounted on the **collection system mount plate** (formerly static mount plate / loaded mount plate).
