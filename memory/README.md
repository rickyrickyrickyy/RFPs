# Marceline — memory (RFP repo)

Marceline’s session summaries and project state live here. When you talk to Marceline in this repo, she loads all `*.memory.md` files from this folder (oldest first).

- **Filename pattern:** `YYYY-MM-DD_short-slug.memory.md`
- **Saving:** Say "remember this" or "save to Marceline's memory"; a new `.memory.md` file is created in this folder.
- **Profile:** `../marceline.md` (Marceline in this repo is standalone; the Agents repo has a full Marceline persona with soul, backlog, and workflows).
