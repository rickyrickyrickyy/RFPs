# Collection subsystem — images

CAD colour coding (includes static deflectors):

| Colour | Hex | Component |
|--------|-----|-----------|
| Bright green | #27C16A | **Collection mount plate** — mounts to frame (loaded mount plate); supports static peelers, juice collectors, filter tubes, plug cutters, static deflectors |
| Reddish-orange | #F36F3B | **Static peelers** — finger-like; interlace with driven peelers to mash fruit; peel squeezed to extraction chamber and outflow chute |
| Yellowish-green | #CCDF3F | **Static deflectors** — keep peels sliding towards the static plate (into disposal channels). Two angled surfaces increase stiffness and may allow forming instead of welding |
| Yellow | #FFFF50 | **Juice collectors (Y-tubes)** — collect juice from filter tube; Y-branch with ribbed connection for standard rubber pipes |
| Dark green | #287F3F | **Filter tube** — juice flows through slots from oranges into yellow collector |
| Light blue | #4DB7F5 | **Plug cutter** — cuts cylindrical plug from centre of oranges during juicing |

## Function (brief)

Oranges fall between **driven** and **static** peelers; fingers interlace and mash fruit. **Static deflectors** guide peels towards the static plate (into separate channels for disposal). **Plug cutter** cuts centre plug. **Juice** → slots in **filter tube** → **yellow collector** → Y-branch → standard rubber pipes (ribbed connection). **Reminder:** Send 34 mm standard connector for quick connection.
