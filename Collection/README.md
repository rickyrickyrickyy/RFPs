# Collection subsystem

**Collection mount plate** (#27C16A), **static peelers** (#F36F3B), **static deflectors** (#CCDF3F), **juice collectors** (yellow Y-tubes #FFFF50), **filter tube** (#287F3F), **plug cutter** (#4DB7F5). Mounts on the frame’s loaded mount plate (collection system mount plate).

## Function

- **Driven + static peelers** mesh; oranges fall between; fingers interlace → fruit mashed, **peel** squeezed into extraction chamber and outflow chute.
- **Plug cutter** cuts a **cylindrical plug** from the centre of oranges.
- **Juice** → slots in **filter tube** → **yellow collector** → **Y-branch** → **ribbed connection** for standard rubber pipes.

**After pressing stroke:** [Plug ejection](../Plug-ejection/) pushes plunger through filter → plug out of plug cutter → plug falls into [disposal](../Outflow-disposal/) core chute. Peels fall to sides of peelers, avoid core chutes, two channels → [disposal](../Outflow-disposal/) (augers take peels and plugs/cores away separately).

## Reminder

**Send the 34 mm standard connector for quick connection** (juice collector outlets).
