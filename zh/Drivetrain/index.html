导言

传动系统将电动机旋转转换成驱动提取循环的回转凸轮运动. WEG W21/W22 IE3电动机(7.5千瓦,1460rpm)通过SPA V带驱动一台Sumitomo Cyclo 6000减速器;减速器输出变为一个螺旋齿轮列车,其凸轮轴承与传输轭接口。 整个组装都挂在专门板块上 螺栓上边框使它成为扭矩、振动和循环装配机器结构的主要来源。

目录链接 : WEG W22 IEC TEFC(二)· 妇女苏米托莫Cyclo 6000目录 (PDF).

讨论情况

粗略设计意图( M)

  • 状态——CAD为传动系统布局和接口而存在,但最终规格(带中心距离,齿轮模块/剖面,螺旋角度决定,详细的轴承选择/安排)仍在开发中.
  • 主移动器——WEG W21/W22 IE3发动机,产品编号1353507:7.5 kW(10 hp),50 Hz,4-pole,额定速度1460 rpm,额定扭矩~49 N~m,38毫米轴.
  • 驱动——SPA带驱动,目前假定比例为1:1(发动机和减压器上170毫米弹管直径拉力),计划设置两个带;中心距离和张力机制为TBD,并驱动显著的射线负载进入安装板.
  • 缩小器——Sumitomo Cyclo 6000系列;输出速度目标~70 rpm. 减速器输出扭矩驱动驱动驱动凸轮轴承的齿轮列车进行对流.

已知挑战/问题和风险

  • 振动 + 循环装配帧——汽车/减速器和腰带张力会引入振动;凸轮对流会增加循环力含量. 关节(bolts + dowels)的设计必须避免滑/滑.
  • 带状张力/射线负载中心距离和张力方法尚未最后确定;带状张力将射线负载设置在运动杆/减速杆和安装板上,并影响承载生命和噪音。
  • 未完成齿轮设计模块(可能为6或8),螺旋螺旋,接触比,硬度/热处理,以及反弹目标尚未最后确定. 这些选择支配着疲劳的生活,噪音,效率和制造能力.
  • 轴承负载浓度——驱动齿轮和胶带滚筒轴承区看到高强度和瞬间,必须携带到传动系统的车架板和车架上而不过度偏转.
  • 轴/骨不匹配 + 自定义扩展SKF挂载轴承装置(例如F4BRP 208单元的标称63.5毫米)不完全符合当前概念中的Cyclo 6000减速轴几何。 减少输入和输出轴都需要延长机器布局。 需要自定义轴延伸/调整器(或备选轴承选择);承包商提出最安全/最易制造的方法。

DFM和制造业(中国)

  • 柴油制造计划- 承包商提出齿轮材料和热处理、牙修补、可实现的耐受性和检查(说明、铅、排出、反弹)。 优先使用中国现有的流程和供应商。
  • 持有安排- 承包商审查定位/不分配办法、预装战略和如何检查/保养。 如果当前 SKF 部件难以在本地来源时提供替代 。
  • 挂载+服务——设计应允许皮带张力调整,拉力对齐检查,减少服务,而不必不必要地去除大型组件.

寻求优化

  • 可靠性——环式装载下齿轮和轴承的疲劳寿命长;强力致错配.
  • 噪声+效率——工业汁线可接受噪声;保持损失可控.
  • 可适用性——快速腰带更换,张力检查,进入润滑点.
  • 减重——如不损及硬度,接触应力,或可制造性,则希望带动齿轮减重.

承包商的挑战/问题

  1. 验证马达/带/减速器的配置,并提出坚固的带张力方法(中心距离,调整范围,目标带张力,以及由此产生的射线负载).
  2. 使用最终齿轮几何,验证传动系统负载箱. 目前的初步数字:减速输出速度~70 rpm;驱动齿轮扭矩~1023 N~m;估计齿轮网力~6.6 kN 微弱和~7.2 kN 正常(待验证).
  3. 建议最后齿轮舱/剖面(spur vs Helical)和热处理/完成疲劳寿命和中国制造。 提供建议的反弹和接触模式目标。
  4. 审查轴承安排(包括用驱动齿轮和凸轮轴承的薄膜滚筒)。 建议替代、预装战略和预期寿命/保养间隔。
  5. 为减压器的输入和输出提出一个完整的轴向解决方案,包括任何必要的轴向扩展/适应器、容量和组装计划。 处理 Cyclo 6000 轴与选定挂载轴承单元之间的比对。
  6. 建议对已建成的传动系统进行验收:反弹范围、排出、对齐、振动/噪音检查,以及简单的负载测试。

供应链和候补

  • 汽车目前规定了WEG电动机;如果替代品可以接受,承包商可提出中方可用的等价物。
  • 缩小器目前假定是Sumitomo Cyclo减量器;承包商提出具有类似性能和足迹的等同物或当地采购选择。
  • 轴心——CAD中存在一些轴承的SKF部分编号;承包商在必要时提出当地可用的等价物。

预期成果和接受

  • 可交付成果最终的传动系统设计包,包括发动机+减压器安装、带式张力法、齿轮图纸/光谱(材料/热处理)、轴承安排和安装板反应的负载情况摘要。
  • 接受情况——传动系统在目标速度上顺利运行,没有异常的振动/噪声;齿轮接触模式和目标内部的反弹;在循环负载下在挂载接口上没有联动滑动.

构成部分摘要

  • 轴心——支持轴旋转,减少摩擦,需要定期检查/加速度来维持寿命.
  • 管道驱动棒将火车/烟囱连接到核心弹出螺旋杆;对齐至关重要,因此杆子可以自由通过线性轴承运行。
  • 轭——加入冲压器驱动棒,将凸轮轴承的旋转运动转换为线性中风进行提取和核心弹出.
  • 插座拉动/驱动括号——将杆与插管连接起来,使所有四个插管保持齐进;需要精确地与滤管和插管对齐.
  • 苏米托莫还原器——降低运动速度,增加扭矩;采用杉间距的升降,润滑,维护等操作手册.
  • 递减器挂载——定位并支持传动系统安装板上的减速器,使马达,皮带和齿轮在负载下保持对齐.
  • 两个齿轮——从减压器向凸轮轴承传递扭矩;间距和反斜线必须设置如此的牙齿共享负载而不绑定.
  • 驱动带和发动机传输——将WEG电动机对准减压器;带张力和对齐应易于检查和调整.
  • 普雷卫士——防止意外接触皮带和牵引装置,应可移动维护,但在正常运行时安全.

接口

  • 输入 :WEG W21/W22 IE3型电动机 → V-belt → Sumitomo Cyclo 6000减压器.
  • 输出 :凸轮轴承接口与轭传输子系统。
  • 挂载 : 传动系统安装板在主帧。
