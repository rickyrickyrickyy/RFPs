传动系统安装盘

部分传送节。 主结构板支持轴架和线性轴承,并在压缩负载和果酱壳下保持对齐.

导言

这个板块挂在框架栏杆上 支撑着35毫米轴向导和线性轴承架 它强烈地影响平行性、约束性风险以及采掘和采掘机之间的一致。 现存两种候选建筑:一是沙刻+机后板,二是焊接+应力-解压+机车板.

讨论情况

粗略设计意图( M)

  • 主要要求- 保持轴平行和对齐,使枷锁不捆绑,即使在冲洗污染和不对称/堵塞装载的情况下也是如此。
  • 两个建筑——(A)沙装+机器后硬度/硬度(B)焊接钢(两块喷水板焊成L+三角沟),然后是应力缓解和机械处理。
  • 卫生表面——板前面部暴露于清洁喷雾/汁液中;纸浆/汁液附近的表面应可清洁(目标粗糙度概念:~RA 0.8μm在关键表面).

已知挑战/问题和风险

  • 扭曲——焊接方案可能需要减轻压力(oven)和巧妙操作,以避免扭曲和失去对齐.
  • 铸造质量- 铸造方案必须考虑到表面质量、肋骨厚度和收缩/薄度风险;需要与铸造商进行协调。
  • 所有定位/快速功能 TBD——道尔方案,螺栓图案,和shim战略没有最终敲定,必须提出可重复重建.

DFM和制造业(中国)

  • 铸币设计说明(当前意图)– 最小壁厚~8毫米;肋骨上的草稿角~3°;钻孔凹陷/地貌包括播后机械布置;分离表面在背面(负-Z)有肋骨.
  • 调整作业——机后上下架面平坦;机器卫生整齐的内表面;机背面的机轴承挂垫;机底浮雕接口用于螺栓升空.
  • 焊接设计说明(当前意图)——两块喷水板焊成L;加入三角沟槽;压力解除后机器临接口/板.

承包商的挑战/问题

  1. 建议铸件+机器或焊接+压力-释放是中国硬度、可重复性、表面质量和周转时间的最佳途径。 提供权衡。
  2. 提议将这个板块挂到框架栏杆上以进行可重复的季节性重建(对齐 螺栓 钻探/梦 dowel工作流程可接受)的dowel/bolt/shim方案。
  3. 确定机械/检查计划,以保证轴/轴一致(基准、扁平/平行目标以及实际检查)。
  4. 建议用于冲洗接触和卫生表面的材料/成品。

接口

  • 输入 :由枷锁/碎片和果酱壳产生的力量/运动通过轴承和承载舱进入这个板块。
  • 输出 :对框架铁路和跨成员的反应;保持对接,以便进行提取/收集剥离器接触。
  • 挂载 :bolts(和计划中的 dowels)为框架铁路;可调整/可变,以实现对齐.
