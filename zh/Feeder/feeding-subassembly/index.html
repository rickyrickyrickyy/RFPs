饲料子组装

进料子组件为:水果摄取系统将一列橙子一并放出来果实支持. 每个单元包括一个安装板、90°宽的入口管、带有弹簧导线的直管和同步弹簧机制——在设计迭代期间,所有装置都向支线车体旋转以方便交换。

说明

进料子组件对入口管的橙子流进行调节,一次释放一次到水果支持上. 提取/同步系统的滚筒推开黄色同步弹簧,允许单橙色下降. 整个子组装都挂在棕色山板上,然后挂在支线上。

议会结构

  • 安装板 ——褐色;底部为子组装;上架为支架体.
  • 进口管组件(合编单元,为安装板):
    • 宽90度入口管 ——绿色;YES的管子,橘子滚到饲料上.
    • 进口管 -粉红色的
    • 进口管安装板 - 浅绿色。
  • 螺纹直管和弹簧导线(合编单元,为安装板):
    • 长发 - 甜点。
    • 直线管 - 紫色的。 - 紫色的。
    • 春季指南 ——深蓝色/teal.
  • 同步弹簧机制——黄色同步弹簧;在提取/同步系统上被滚筒推开,将橙色投向果实支撑. 同步弹簧安装到2×深绿色同步弹簧挂在两个粉红色滚筒上.

颜色密钥( 本子组装的数字)

仅适用于本节中的 CAD 视图。 其他子系统使用自己的颜色集.

颜色构成部分

春季尺寸设计参数

  • 弹簧放置在45°角到管(管中心前方110毫米,后方250毫米).
  • 水果支持推杆在最靠近点处离管5毫米处放置以防止碰撞。
  • 推力将弹簧5毫米通过管外侧.
  • 推进器射程178毫米,由870个传动系统确定.

春季规格

  • 静态长度=250毫米
  • 直径=2毫米
  • 外径=22毫米
  • 在静态位置,弹簧长250毫米;在拉伸位置,长306毫米.

90°管道参数

指圆形90°入口管。 。 。 。

  • 标准 :3′′ 核动力源 / 90 DN 管道,附表5S——制造商应当选择在中国拥有的类似规格的任何产品.
  • 粮食区流动限制:目标内直径为~90毫米口径(选择符合内流直径要求的中国现有最接近的管道)。
  • 尺寸 :88.9毫米OD,壁厚2.11毫米.
  • 焊接法兰的分档索引.
  • YES管在62°处切割.
  • CIP喷雾喷嘴的垂直切除——包含在最新的CAD中;照片/图应该刷新以显示最后喷嘴位置.

接口

  • 输入 :豆瓣入口管(feeder top-level)的橙子滚入90°入口管.
  • 输出 :滚筒上提取/同步系统推开黄色同步弹簧;一个橙色滴到水果支持上.
  • 挂载 :子组装安装板附着在支架上.
  • 下游 :取出后,果汁和皮条流入收藏系统(管道、过滤管)。

水果放置护栏(左/右管对齐)

  • 核查左/右管位置是否将果实直接放入正确的铁路/着陆区果实支持。 。 。 。
  • 将管的横向放置作为功能耐受性,因为放置不当会增加干扰/凝胶风险.
