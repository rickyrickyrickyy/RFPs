简介

果芯弹出 system: 柱塞驱动支架, plungers, and the plunger drive system in the side panels; ejects the core from the plug cutter into the disposal core chute.

颜色说明与部件

Plunger assembly, drive bracket, juice outflow — detail pages TBD. Green housings enclose spring and guide rods; interface with filter/plug-cutter and core chute.

颜色部件
Plunger — pushes through filter to eject core (four in parallel)
柱塞驱动支架 — carries plungers; connects to drive rods
Plunger drive rods — linear actuation; through LMK16 into green housings
LMK16 linear bearing — guides rods at housing interfaces
Plunger drive spring — return force; coiled around rod in housing
—Filter — plunger passes through; juice to 收集. Plug cutter — in 收集; core ejected by this system

讨论

果芯弹出 (cycle)

  1. Pressing stroke: Juice to collector Y-section; core remains in plug cutter / filter.
  2. 果芯弹出: Plunger pushes through filter, ejects core from plug cutter → core falls into core chute (disposal).
  3. 废物处理: Peels to sides of 削皮器; two augers take peels and cores away separately.

接口

  • Input: Core in plug cutter/filter after pressing stroke (from 收集).
  • Output: Core ejected → falls into disposal core chute.
  • Drive: Plunger actuation from drivetrain / 传动 (yoke/cam).
