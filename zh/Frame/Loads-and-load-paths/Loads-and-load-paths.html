返回框架· P6 RFP 概览

载荷路径

本页介绍在P6框架中操作的主要操作负载——来自传动系统, 传输,提取,和核心弹射子系统——以及这些负载是如何通过焊接的结构,安装板,以及螺栓的关节下至平脚和地板的. 它属于框架为承包商验证框架的硬度和一致性提供了负载依据。

主要负载

框架必须携带来自传动系统, 传输, 提取(驱动箱和皮革),以及核心弹出没有过度偏移。 必须在枷锁、轴承和剥皮机之间保持对齐,以保持时间和轴承寿命。

来源装配类型通过帧路径
传动系统(机动车、减压车、齿轮)驱动扭矩的反应;发动机、减压器、齿轮列车的重量传动系统安装板 → 侧轨 → 主框架 → 平整支座 → 地板
传输(烟囱、轴承、轴承)对线性中风力的反应;枷锁和驱动质量的惯性传输安装板 → 侧轨 → 主框架 → 平整支座 → 地板
采掘(驱动箱、剥皮机)汁液喷射时的压抑力;静电剥离器的反应装载安装板(收集侧) ↔ 驱动支架; 装载板 → 侧轨 → 主框架 → 地板
核心弹射( Plunger 驱动器)对冲力的反应(压缩弹簧、棒负荷)柱塞外壳和支架在装载/收集结构中反应; 在装载安装板和侧轨中 → 主框架 → 地板

关键负载大小写( 要验证)

下面的目前数值是初步的,旨在帮助承包商确定装载量和接受指标。 订约人用最后几何、带状中心距离、齿轮舱/剖面和凸轮/焦距细节进行验证。

  • 汽车/腰带/减速器——WEG W21/W22 IE3型电动机(7.5 kW / 10 hp)通过SPA带(ratio目前为1:1;计划为两条带)驱动减速器. 带状张力和电动机振动将射线负载和振动传递到传动系统安装板和框架.
  • 减少输出/齿轮网——目标减速器输出速度~70 rpm. 初步传动系统计算显示驱动齿轮为~1023 N~m 扭矩,估计齿轮网状力为~6.6 kN直径和~7.2 kN正常(待验证).
  • Cam-yoke 重置——凸轮轴向轴线偏移~89毫米,驱动枷锁对流. 这引入了循环力和瞬间 进入传输板, 两个驱动轴, 及其支撑。
  • 峰值压缩事件——最坏的操作条件假设最大传动系统能力驱动提取/压缩事件;帧和安装板应以适当的安全因素承受,且不失去对齐性.

线性轴承和安装板上的时刻

传输采用两个轴向,由两个LMK35U(35毫米)线性轴承制导. 轴承位于安装板上方,因此离轴力和左/右两侧之间的任何不对称都会产生进入传动系统安装板和框架的瞬间.

  • 非对称/ 干扰大小写——如果一方绑定或看到比另一方更高的摩擦,竖杆可以扭动,将额外的瞬间负载加在线性轴承和板上.
  • 倒转的时刻——轴承反应线与安装板之间的垂直抵消产生一个倾覆瞬间,倾向于在水平轴上旋转安装板. 在压缩过程中,这在收藏方面预计会很显著.
  • 装配传输——这些时刻必须从安装板上传入道巾/排弹针,并用螺栓夹住侧轨,然后传入焊接框架,然后下穿平面山和地板.

载荷路径原则

  • 清除路径——任何单一成员或关节都不应看到过重的负载;负载应分散在焊接结构中,并按原意排入三个板块.
  • 最小化偏转——主框架和板块接口的硬度至关重要,这样,枷锁行走,剥网,和螺旋对接都停留在设计意图之内.
  • 地面反应——所有操作负载都必须通过腿向下承载,进入平面架(M24端板)和地板;没有浮动或罐装的反应路径.
  • 关节——包装接口(套装到框上,括号到板块上)必须足够用于剪切和张力;毛巾携带位置和剪切位置指定.

RFP的说明

要求承包商核实和记录已建配置的装货案例和装货路径,并提出硬度和对齐的实际接受目标(包括输电井的不对称/侵入案例)。 承包商应酌情建议简化计算是否足够,或是否有必要对框架焊接和(或)关键的安装板进行FEA。

承包商的挑战/问题

  1. 衍生验证的负载箱(运动器/带状光圈负载来自张力,齿轮网力,凸轮对流力,峰值压缩),并在每个安装板接口提供力/运动估计值.
  2. 验证或纠正初步驱动装置的数字(按驱动齿轮计算为~1023 N·m扭矩;~6.6 kN切变和~7.2 kN正常齿轮力)和文件假设.
  3. 对LMK35UU制导轴的瞬间和轴承负载进行量化,包括一个非对称/遮挡箱,并建议安装板硬度和联合设计。
  4. 建议是否需要FEA(机身焊接、磁带滚轴承周围的传动系统板、线性轴承周围的传输板)并提供边界条件和验收指标。
  5. 提出可允许偏移/偏移的接受指标,以保持剥离器的对齐和承载寿命,并建议检查以核实。
