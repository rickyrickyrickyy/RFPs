机器控制定义和UI

部分管制和电子节。 地方独立控制,指标,安全UI,以及停机/停机.

导言

本页面定义了操作器的控件和P6在管制和电子子系统。 机器必须作为具有本地ON/OFF的独立单元运行,并在顶端运行/停止控制,有明确的RUNNING/STOPPED指标和安全开放状态. 三个编码盖安全间锁(来自附文/壳当任何封面打开时防止操作。 需要后关/停机装置(LOTO)来提供服务和季节性拆卸。

讨论情况

粗略设计意图( M)

  • 独立运作——局部动力和运行控制允许独立运行,检查,更换部件,以及简单的清洁.
  • 清除状态指示——操作员应当迅速看清机器是否因安全开放条件而运行,停止运行,或者无法运行.

控制面板要求(前期)

  • 启动/启动密钥切换
  • 运行/被支持按钮
  • 紧急停车按钮
  • 运行/部署指标
  • 运营/业务指标
  • 机器打开/ 安全开口指标( 间隙状态: 任何支线/ 插头/ 驱动列车盖盖开口)

LOTO 要求(上-后)

  • 方便获取——迅速隔离电力服务.
  • 可锁定用于停产/停产过程(种植程序可能不同)。

已知问题和风险

  • 冲洗接触——UI和布线必须防水;选择适合工业清洁的围网,封条和腺体.
  • 安全逻辑的明确性——避免模棱两可的状态;明确机器为何不会启动(打开盖子,E-stop等).

承包者的问题

  1. 建议适用于冲洗接触的工业部件和附件/IP评级。
  2. 提出具有明确标签和指示逻辑的工学(顶部)和可维护的面板布局。
  3. 提出符合植物实践的LOTO架构和位置,同时对操作员来说仍然简单.
  4. 建议采用与商用机器相同的控制板,采用编码安全盖硬件(磁铁码防误用)和布线/集成方法。

接口

  • 输入 :操作员按钮按下; 安全间隔状态 。
  • 输出 :运行/停止命令到运动控件;状态可见指示器。
  • 挂载 :前置面板; 连接到内部控制柜的线路。
