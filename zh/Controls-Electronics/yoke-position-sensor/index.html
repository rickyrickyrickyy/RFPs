Yoke 位置传感器

部分管制和电子节。 循环/烟雾感测安全停机条件和循环计数(TBD实施)。

导言

枷锁位置传感器——其中一部分管制和电子子系统 —— 通过感知在传输。这支持三种功能:(1) 控制的安全停止条件(在不安全的状态下不要用剥皮器停止),(2) 周期计数操作监测,以及(3) 一种关键的"完全开放的停止"行为,这样操作员可以在机器正常关闭时服务或移除内部部件.

准确的传感器类型(编码器,距离传感器,近距离开关等)是TBD. 传感器硬件的实施预计将涉及ODM/供应商,例如:专业。 。 。 。

讨论情况

粗略设计意图( M)

  • 停止与 pelers 打开间锁——在剥皮机打开/不安全状态时,传感器/日志应防止操作(或强制停止).
  • 周期计数- 为行动监测提供每个周期的清点。 可能与安全停止逻辑共享传感器硬件.
  • 正常关闭时完全开放停止——当机器在正常运行中被关闭/停止PED(不是E-stop,不是盖-interlock断层)时,驱动列车应该停止,枷锁必须结束在完全打开位置因此,可以安全地去除螺旋桨/螺旋桨相关部件。
  • 紧急停止/ 交叉行为——当按下E-stop,或者当一个编码的盖子安全开关表示开盖时,机器必须立刻关闭(没有重新定位到打开的要求).

已知问题和风险

  • 冲洗——感应器和电线必须具有防水性能和固态以清洁化学品.
  • 信号完整性——振动和污染可造成假触发;选择传感器类型并相应加装.

承包者的问题

  1. 建议采用在冲洗/振动中坚固的传感器方法(编码器与近距离与极限)和安装。
  2. 定义传感器将如何用于安全停止和循环计数,包括故障检测(故障安全行为).
  3. 提出确定完全打开枷锁端位置(远程传感器目标对旋转编码器参考标记),以及维护后如何进行校准.
  4. 描述传感器信号与现有控制板的接口,以及J-Pro(ODM)如何/何时提供感知解决方案。

接口

  • 输入 :机械运动(yoke/cam/shaft位置)和安全开关状态。
  • 输出 :循环计数和/或位置信号用于控制器;在不安全时抑制运行。
  • 挂载 :接近驱动电车/传输机制;有线线路通往控制柜。
