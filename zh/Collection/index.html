导言

收集子系统是提取组装的静态一半. 它装有静电剥皮机、滤波管、插头切割器和果汁收集器(Y-tube),全部安装在收集器的挂板上,这些挂板将螺栓栓栓在装车的挂板上。 在开采过程中,提取同步将果汁与果皮和种子分开 汁液通过滤波管内的槽流入收集器,并通过卫生的快速连接管管流入第三方罐体;核心留在滤波管内,以供喷射核心弹射系统。 。 。 。

讨论情况

粗略设计意图( M)

  • 这个子系统做什么——提供静态提取半(静态剥皮机),并通过滤管收集汁液进入集热器Y-tube,然后将汁液输送到第三方罐/泵. 同时整合插头切割器,支持核心弹出通过滤管.
  • 汁收集器管概念——当前概念为中国可用使用标准不锈管尺寸:主采集体~57毫米OD(选择为标准尺寸;不严格要求)和汁液排出管大小,以接受快速连接硬件(当前概念:~38毫米OD排出通过肋骨连接器连接橡胶软管).
  • 过滤管——结构上与商用机器相似但更长;计划为硬化钢,有激光剪切的垂直槽(~0.7毫米槽宽概念)和清除纸浆每个循环的刮片机/板机接口.
  • 收藏挂载板——与传输挂载板相同的整体想法:僵硬,对齐关键,可清理. 内表面完成目标高(Ra~0.8μm概念),必须支持剥离器和采集器对齐.

已知问题和风险

  • 喷管/滤管密封泄漏——已知问题:果汁漏出过冲压器到滤泡器的接口;预计在更高的工业压力下会恶化. 可能需要O环或替代海豹策略.
  • 穿着——前置配置有304个刮纸机,与304个滤管相对应,对管ID进行粗糙. 目前的意图是硬化钢材用于改进磨损;承包商负责验证。
  • 过滤管对齐敏感度——滤管焊接到一个盖子上,然后组装(挤压)到位置;堆叠式可以移向对齐,使螺旋管密封和磨损恶化.
  • 碰撞风险- 这个驱动槽偏转器必须坚固、无碰撞和清洁。 它在压缩时必须覆盖核心槽,同时仍然允许CIP喷雾按设计到达周围/穿过.

DFM和制造业(中国)

  • 钢管制造——优先使用中方提供的卫生/快速连接配件进行标准管尺寸和焊接组件.
  • 硬化管加工- 承包商提议如何制造、硬化和完成(ID研磨)滤波管,同时保持槽几何准确和清洁。
  • 卫生——接触果汁的内表面应可清洁,光滑,避免裂缝(用承包商和卫生规则确认最终目标).

CIP/清洁概念

  • T-valve 模式开关——概念:在将果汁送入储油罐与将CIP液喂入集热器输出口之间,使用T-阀交换.
  • 逆流清理——CIP液通过排水管进入,流入收集器/汁室,然后通过过滤器管向上,然后通过插头切开器中央排出,以冲槽和清洁内部.

承包者的问题

  1. 提议一种强力的密封策略,用于冲压器到滤泡管,以减少高压下汁液的泄漏(O环、唇印、收紧尾/耐受性、替代几何)。
  2. 建议材料 + 热处理 + 完成过滤管和插件切割器,以管理刮刮器的磨损并保持卫生。
  3. 提出一种组装和对齐方法,使滤波管和冲压器对齐保持稳定(特别是给定的"带盖+螺丝英寸"堆栈).
  4. 验证CIP逆流概念(T-valve切换,流路,清洁有效性),并提出任何所需的检查阀/过滤器.
  5. 建议使用中国常见的卫生部件的采集器管和快速连接接口的制造战略。
