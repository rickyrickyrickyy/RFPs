项目概览

P6工业原型和RFP结构的单一参考.上次更新 :2026-03-20 (中文(简体) ).

P6型是德利胡斯线上最新一代的柑橘汁机. 它旨在处理多种柑橘品种(包括柠檬),具有可重复的工业提取循环,将果汁与废流分离. 方案正在建立一个试点生产环境江西(中国)目标设置10月(半天会议)第一台原型和验证的机器是第6-870次会议; 数据P6-970 (英语).3x较大的提取模块计划稍后进行. P6-870 吞吐量:~280 水果/min,估计产量1360升/小时果汁的饮料。 设想试验线6个系列提取器到达~8160升/小时,所以所有下游第三方设备必须达到总容量。 P6-870型机车的设计目的是与现有的工业橙汁线结合,可替换为:JBT 马雷尔风格由开义或类似的线路提供商制造的机器.

P6是怎样的?

临6是当前工业原型柑橘机 围绕Delijuice 提取周期(功能序列与之前的Delijice工业提取器相同).机器线条 :P3 + P4 → P5 →临6(今天的发展重点). P6方案正由P6方案合作进行。丢弃和是 弯曲(编). 小一点323国道/ 逆顶线是一个单独的产品家族——不是这个RFP的主题.

  • 江西(JX):工厂长期协议(长泰安)最小植株目标:8,160升/小时,6个提取器,涡轮滤波器,消毒器.行动压力:10月。
  • 绘图/ fab 目标 : 5月15日,日内瓦(深圳地区)

机器如何运作

Delijuice提取周期(高级)
  1. 水果摄入量——柑橘一次上演,呈上提取腔.
  2. 压缩——果实压缩/浸泡;果汁通过过滤/收集路径传递.
  3. 汁液外流——汁液流入收集器,然后流入罐体/净化设备.
  4. 核心驱逐——插件/核弹弹出并通向核心废物槽/电机。

关于中风时间, 枷锁运动, 以及摄入与压缩的相互作用,Delijuice 提取周期。 。 。 。

工厂产品流动(果汁)

上游 → P6 → tank tank 下游.第三方分拣和饲料给P6进入管带来水果;P6输出果汁给植油槽;剥皮和芯片去分开的Augger线.

  1. 第三方排序 + feed——通过大小分拣器将预置,预洗的柑橘进入P6饲料线和入口管.
  2. P6 提取分阶段水果被剥皮/压缩;果汁通过过滤/收集;剥皮和核心以分离废物槽/auger。 ~1360 L/h/每个提取器;规划中的6×线~8160 L/h.
  3. 汁罐头- 从收藏家到果汁罐头总行率的大小。

总体设计、制造和食品安全要求

材料、食品安全和表面

  • 在所有食物和溅射区(如不锈钢和相容聚合物)使用食品安全材料;避免粘合物成对和腐蚀性完成.
  • 在食物区,目标表面粗糙度在Ra 0.8-1.6微米左右,因此表面是清洁的,不会夹住纸浆.
  • 遵循NSF式的指引:食物或喷洒区没有架子,口袋,或隐蔽区;所有食物和液体都必须有明确的排水路径.
  • 密封或焊接食物和溅射区的所有缝隙,从而没有毛细缝隙;避免在这些地区出现腿关节和未密封的重叠。
  • 在所有边缘;没有尖锐的边缘或凹槽暴露在操作员或任何能够收集食物的区域。

栅栏和关节

  • 最小化不同紧身尺寸的数量;在实际限制在M3,M5,M10,和M16以提升强度的情况下.
  • 在食物接触区:无曝光线索使用对峙,焊接螺旋, 或通过螺旋 与坚果在区外。 如果需要机器螺丝来服务,菲利浦斯,六指头,还有时间档驱动器是可以接受的;除非商定,否则在食物区或食物区以上避免其他驱动器类型。
  • 在剪切中定位部件的首选圆柱;在带剪切和旋转的关节使用肩螺栓.
  • 最小化带有共享括号和清晰负载路径的全加固器计数;设计方便拆卸和重新组装用于清理.

制造设计

  • 偏好对称部分,尽可能避免不必要的交接.
  • 通过使用库存尺寸、简单的焊接和激光/颗粒剪切图谱,将机械加工和二次加工最小化。
  • 指定足够函数的容积;后处理方法(如磨磨,磨磨,磨磨)由制造商选择以满足这些容积.
