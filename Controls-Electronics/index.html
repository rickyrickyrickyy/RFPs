11. Controls and electronics

Subpages

Machine control definition and UI, Motor power management, Yoke position sensor (TBD) — content on this page.

Introduction

User interface, safety, motor power, and yoke position. The subsystem consists of the control panel, safety switches, motor power electronics, and the yoke position sensor (under development).

Discussion

User control panel

Operator interface for power, run control, and status.

  • Emergency stop — hard stop; machine cannot run until reset.
  • Machine on/off switch — main power to the machine.
  • Machine operation start/stop button — run/stop the extraction cycle.
  • Indicator lights — machine open (when any of the 3 lids is open), plus indicator(s) for other sensor status as needed.

Safety switches

Safety switches on 3 lids (e.g. plunger cover, drivetrain cover, feeder). When any lid is open, the machine shuts off and cannot run. Interlock logic must be fail-safe.

Motor control and power management

AC power distribution and motor drive for the drivetrain motor.

  • AC contactor — main power switching for the motor circuit.
  • Motor starter — start/stop and protection for the main drive motor.
  • Wires and cable management — routing, glands, and management as needed for control panel, motor, and safety circuits.
  • Motor: WEG W22 — spec sheet to be provided; document power rating, voltage, speed/torque, and mounting for the RFP.

Yoke position sensor

Encoder or other means to detect the position of the yoke in the cycle. Required to know where the yoke is for controlled stop position and cycle logic. TBD — needs further development.
