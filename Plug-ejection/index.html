Introduction

The core ejection system (subsystem 7) removes the pressed orange core from the plug cutter and drops it into the disposal system's core chute. It uses compression-spring-driven plungers mounted on a drive bracket, actuated through linear-bearing-guided drive rods housed in the side panels. The transmission yoke pulls the drive rods back against the springs during the return stroke; on release, the springs push the plungers forward through the filter tube to eject the core.

Discussion

Core ejection (cycle)

  1. Pressing stroke: Juice to collector Y-section; core remains in plug cutter / filter.
  2. Core ejection: Plunger pushes through filter, ejects core from plug cutter → core falls into core chute (disposal).
  3. Disposal: Peels to sides of peelers; two augers take peels and cores away separately.

Interfaces

  • Input: Core in plug cutter/filter after pressing stroke (from Collection).
  • Output: Core ejected → falls into disposal core chute.
  • Drive: Plunger actuation from drivetrain / transmission (yoke/cam).
