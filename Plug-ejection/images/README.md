# Plug ejection subsystem — images

Components visible in CAD: **plunger actuator bar** (light purple), **plungers** (blue, multiple stations), **plug ejection channels/housings** (green), **actuator rods** (red, yellow), **guides/bushings** (orange, light blue), **return spring** (yellow, in housing). Plungers push through filter and plug cutter; plugs fall into [disposal](../Outflow-disposal/) core chute.
