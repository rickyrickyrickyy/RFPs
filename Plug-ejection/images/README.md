# Plug ejection subsystem — images

CAD colour key: **plunger** #5151FA, **plunger drive bracket** #D1A4FF, **plunger drive rods** #B31B5B, **LMK16 linear bearing** #EBA13D, **plunger drive spring** #8CC135. Green housings enclose the spring and guide the rods. Plungers push through filter and plug cutter; plugs fall into [disposal](../Outflow-disposal/) core chute.
