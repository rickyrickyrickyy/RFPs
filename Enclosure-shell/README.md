# 9. Enclosure / shell

**Related to:** <a href="../Frame/">Frame</a>. Protective covers over moving parts; overcentered hinges for feeder, drivetrain lid, and plunger lid.

## Protective covers

- **Plunger cover** (#BC4944) — red; protects the [plug ejection](../Plug-ejection/) plungers.
- **Drivetrain cover** (#489C26) — green; covers the [drivetrain](../Drivetrain/).

## Overcentered hinges

The **feeder**, **drivetrain lid**, and **plunger lid** are all mounted using **overcentered hinges**. These hinges keep the lid in the open position when opened and keep pressure down when the lids are shut.

**WIP:** The **springs** and **assembly method** for these overcentered hinges are **not designed yet**. Details to be provided once specified.
