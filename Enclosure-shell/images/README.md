# Enclosure / shell — images

Ten CAD views of the protective covers (plunger cover, drivetrain cover) and overcentered hinge mounting.

## CAD colour key (this section’s figures only)

| Colour (hex) | Component |
|--------------|-----------|
| **#BC4944** | Plunger cover — red; protects plungers |
| **#489C26** | Drivetrain cover — green; covers drivetrain |
| Blue | Overcentered hinge — linkage/bracket; mounting to covers and frame |
| **#E3B938** | Base / frame — yellow |

## Notes

- **Overcentered hinges:** The feeder, drivetrain lid, and plunger lid are all mounted using these hinges. They keep the lid in the open position and keep pressure down when shut. **Springs and assembly method are not designed yet.**
